図2.2　SLO設定の4つのシナリオ

2軸のマトリクス図で、各象限にシナリオ1〜4が配置されている。

```
                行動：
              信頼性向上を優先
                  │
     シナリオ1（○） │ シナリオ2（×）
                  │
信頼性：不満 ────────┼──────── 信頼性：満足
                  │
     シナリオ3（×） │ シナリオ4（○）
                  │
                行動：
              機能開発を優先
```

各象限の読み方：
- シナリオ1（左上）：信頼性に不満あり → 信頼性向上を優先（○ = 適切な対応）
- シナリオ2（右上）：信頼性に満足あり → 信頼性向上を優先（× = 不適切な対応）
- シナリオ3（左下）：信頼性に不満あり → 機能開発を優先（× = 不適切な対応）
- シナリオ4（右下）：信頼性に満足あり → 機能開発を優先（○ = 適切な対応）
